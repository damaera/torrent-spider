# TorrentSpider

Web app for https://torrentspider.ga

Search and magnet torrents from multiples providers
- ThePirateBay
- KickassTorrents
- TorrentProject
- Rarbg
- Torrent9
- Torrentz2
- 1337x
- ExtraTorrent


## Install

`yarn install`

## Running development server

`yarn dev`

## Build

`yarn build`
